# Smart Health & Medical History Tracker + Wellness Analytics System (SHM-WAS)

This project is a Python-based system for managing and analyzing personal health data, including patient records, vitals, and medical history, with support for data storage (JSON/CSV), analytics (NumPy, Pandas), and visualization (Matplotlib) through a modular, OOP-based architecture.
